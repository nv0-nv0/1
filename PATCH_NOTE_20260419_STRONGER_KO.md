# NV0 추가 보강 패치 메모

이번 보강에서는 단순 문구 수정이 아니라 관리자 인증 흐름 자체를 다시 잡았습니다.

## 반영 내용
- `start_server.py`에서 `.env`, `.env.local`, `.env.production` 자동 로드
- `server_app.py`에서 동일 환경파일 자동 로드
- 관리자 세션 쿠키를 요청 스킴 기준으로 발급
  - HTTP 로컬 실행 시 `Secure` 미부착
  - HTTPS/프록시 환경에서는 `Secure` 유지
- 이미 로그인된 상태에서는 관리 버튼 클릭 시 키 재입력 없이 바로 진입
- 관리자 상태 동기화 실패 시 메뉴 자체를 숨기지 않도록 변경
  - 인증 성공 후에는 운영 메뉴를 즉시 표시
  - 동기화 지연 메시지만 안내

## 직접 해결한 증상
- 관리 키 입력 후 다시 키 입력 화면으로 돌아가는 문제
- 이미 인증됐는데 메뉴 진입 때마다 또 키를 묻는 문제
- `.env` 미로딩으로 관리자 키가 의도와 다르게 동작하는 문제
- 상태 동기화 실패를 인증 실패로 잘못 표시하던 문제

## 로컬 확인 포인트
1. `python start_server.py` 실행
2. `.env`의 `NV0_ADMIN_TOKEN`으로 로그인
3. `/admin/index.html` 진입 후 메뉴 노출 확인
4. 상단/사이드 `관리` 버튼 재클릭 시 재입력 없이 진입 확인
