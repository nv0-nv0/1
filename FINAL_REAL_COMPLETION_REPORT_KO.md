# NV0 최종 완료 보고서

## 최종 판정
NV0는 **즉시 시연 + 결제 진입 + 결제 후 자동 발행 + 고객 포털 납품 확인 + 게시판 운영 + 관리자 허브 + board-only 전환**까지 포함한 납품 패키지 기준으로 완료되었습니다.

## 이번 납품본에서 정리한 것
- 전체판(full mode)을 기본값으로 복원
- board-only mode를 옵션으로 유지
- 문서 전부를 실제 현재 구조에 맞게 정리
- 납품용 패키지 생성 스크립트 추가
- 배포 후 스모크 테스트 스크립트 추가
- Docker 빌드 컨텍스트 정리
- 최종 패키지 인벤토리/해시 자동 생성 체계 추가

## 기능 완료 범위
### 사용자 측
- 제품 목록/상세 페이지
- Veridion 즉시 시연
- 체크아웃
- Toss 결제 승인/실패 페이지
- 고객 포털 조회
- 게시판/문서/가격/사례/FAQ

### 관리자 측
- 상태 확인
- export/import
- 즉시 발행 / 재시드 / 초기화
- 샘플 주문 생성
- 주문 재발행 / 단계 진행 / 결제 토글
- 백업 / verify-only 복구 검증

### 운영 측
- full mode / board-only mode 전환
- startup auto-build
- 패키징 런타임 검증
- 동시성/중복 결제/웹훅 재전송 방어

- full API E2E 통과

## 아직 외부 조건이 필요한 마지막 확인
