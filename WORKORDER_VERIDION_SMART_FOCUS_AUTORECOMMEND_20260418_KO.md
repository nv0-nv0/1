# Veridion 스마트 추천 패치

적용 항목
- 주요 운영 국가 드롭다운 기본값 대한민국 유지
- 업종 + 운영 국가 기준 중점 확인 사항 자동 추천
- 사이트 주소 도메인 기준 운영 국가 자동 반영(.kr/.jp/.cn/.eu/.com 등)
- 사용자가 중점 확인 사항을 직접 바꾸면 수동 선택 유지
- 일반 데모 / 제품 데모 정적 페이지 동시 반영
- 소스 및 배포 파일 동시 반영

추천 기본값
- 이커머스: 결제·환불·청약철회 고지
- 뷰티·웰니스: 광고·표시 문구
- 의료·건강: 민감 업종·표현 위험
- 교육·서비스: 약관·정책 문서 일치
- B2B SaaS: 개인정보·회원가입 동선
- EU 운영 시 개인정보 중심 추천 우선 반영
