# NV0 최종 작업지시서 및 완성 패키지 납품서

## 1. 남은 단계 전수 파악
현재 기준 남은 단계는 총 3단계입니다.

### A. 패키지 내부 작업
- 남은 단계 수: 0
- 판단: 완료
- 근거: UI/UX 개선, 홈/제품 전환 보강, 데모 업셀 블록, 관리자 운영 가시화까지 패키지 내부 수정 반영 완료

### B. 운영 반영 전 최종 검증
- 남은 단계 수: 1
- 항목: 실제 운영 도메인 재배포 후 정적 캐시/라우팅/환경변수 반영 확인
- 비고: 패키지 밖 운영 환경 단계

### C. 실결제 검증
- 남은 단계 수: 1
- 항목: Toss live 키 반영 후 실제 승인-전달-포털 연결 확인
- 비고: 외부 결제 키 필요

### D. 외부 연동 검증
- 남은 단계 수: 1
- 항목: Veridion 법령 API 또는 실제 외부 점검 대상 사이트 기준 live 품질 확인
- 비고: 외부 대상 정책/응답 상태에 영향 받음

## 2. 이번 최종 패치 범위
### 2-1. 홈 화면
- 샘플 리포트 느낌을 먼저 보여주는 3단 카드 추가
- 전환 전 신뢰 형성용 설명 강화
- 기존 세로 위주 인지 흐름을 더 수평 카드형으로 정리

### 2-2. 제품 상세 화면
- 핵심 메뉴 아래 샘플 전달 화면 카드 추가
- 추천 플랜/시작가를 한 번에 보는 비교 레일 추가
- 데모 이후 결제·전달·포털 연결 흐름 카드 추가

### 2-3. 데모 결과 화면
- 결과 확인 직후 바로 이어지는 업셀 블록 추가
- 추천 플랜 보기 / 가격·결제 바로가기 CTA 연결 강화
- 저장형 결과와 비저장형 결과 모두 같은 흐름으로 이동 가능하게 정리

### 2-4. 관리자 화면
- 관리 보안 상태 카드 추가
- 결제 런타임 상태(Mock/Live/Off) 가시화
- 자동화 상태 카드 구조 정리

## 6. 운영 반영 작업지시
### 1단계. 배포 환경변수 확인
필수 권장값
```env
NV0_ADMIN_TOKEN=32자이상_랜덤문자열
NV0_REQUIRE_ADMIN_TOKEN=1
```

실결제 운영 시 추가 확인
```env
NV0_TOSS_CLIENT_KEY=실키
NV0_TOSS_SECRET_KEY=실키
NV0_TOSS_MOCK=0
```

### 2단계. 재배포
- Git 반영 또는 패키지 업로드
- Coolify Environment Variables 저장
- Redeploy 실행
- 강력 새로고침 후 공개 화면/관리 화면 재확인

### 3단계. 배포 직후 점검 URL
- `/`
- `/products/veridion/index.html`
- `/products/veridion/plans/index.html`
- `/products/veridion/demo/index.html`
- `/board/index.html`
- `/admin/index.html`

### 4단계. 합격 기준
- 홈에서 샘플 리포트 카드 3개 노출
- 제품 상세에서 샘플 전달 화면 / 플랜 비교 레일 노출
- 데모 결과 후 추천 플랜 / 가격·결제 바로가기 노출
- 관리자 비인증 시 로그인 게이트 유지
- 관리자 인증 후 보안 상태 / 결제 런타임 카드 확인
